# Chat

You can chat with website visitor from here. Any new visitor ask you to something from the bottom right chat icon you will get those visitor messages from here. You will be able find the Chat page on the following admin link.\
`http://website.com/dashboard/chat`

### List <a href="#list" id="list"></a>

From the left sidebar menu you need to click on the **Chat** section to go to the Chat page.

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FA1I7EWEQvfGRplw7PX3X%2F22_chat_list.png?alt=media&#x26;token=fe808dfc-1f9a-4e79-a5ea-c34871b9527a" alt=""><figcaption><p>Chat</p></figcaption></figure>

You will find all of the conversation with the customer name and email if you look at the left sidebar.

Clicking on a conversation you will see existing histories if have. You can type on the message input box and talk with customer realtime.

### Delete Chat <a href="#delete" id="delete"></a>

You can delete an existing conversation easily with clicking on the right top delete icon

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FIExXI7Cwk2PTsOpPNqn3%2F23_chat_delete.png?alt=media&#x26;token=fa2e3a4f-f285-4f7b-a75c-34291d12b01f" alt=""><figcaption><p>Delete Chat</p></figcaption></figure>

\*\*\* Deleting a chat/conversation will not delete permanently, it will just delete from the list and you will find that if you click search left filter section, `Filter > Trashed > With Trashed / Only Trashed`
