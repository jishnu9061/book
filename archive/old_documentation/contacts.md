# Contacts

Contacts is an additional module where you can add your essential contacts that you need. You would be able to manage your contacts here. You will find that contacts page on the following URL. `http://website.com/dashboard/more/contacts`

### List <a href="#list" id="list"></a>

You will find the contact list page if you click in the `Contacts` menu from the left sidebar section.

### Create <a href="#create" id="create"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FYXkP8aNfNDQWK8AjpdLV%2F48_contacts_new.png?alt=media&#x26;token=d43ee921-2198-4b66-a684-57b5338f9c87" alt=""><figcaption><p>Create a new contact</p></figcaption></figure>

You would be able create a new Contact item if you click on the `Create New` button on the Contact list page.

### Update <a href="#update" id="update"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FduitwMYmeinhjLqL6EF6%2F46_contact_update.png?alt=media&#x26;token=b966f70a-eeb4-45c1-800c-11815980744f" alt=""><figcaption><p>Update contact</p></figcaption></figure>

You can update an existing contact item easily with select/click on any contact item from the contact list page.

### Delete <a href="#delete" id="delete"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FbYyicts17cFLhNAbfwXy%2F47_contacts_delete.png?alt=media&#x26;token=78a3621e-a71b-4fad-8f8d-39f6eceb8f19" alt=""><figcaption><p>Delete a contact</p></figcaption></figure>

Go to the contact details page with clicking on a particular Contact item. Then click on the `Delete Contact` button to delete that particular contact.
