# FAQ

FAQ is a feature where define lots of questions and answers that customer might have. On the Dashboard you would be able to set those question and answers easily. You will be able find the FAQ manage page on the following admin link.\
`http://website.com/dashboard/faqs`

### List <a href="#list" id="list"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FgM2nTCmH3SqWq4Nur5ZI%2F25_faqs_list.png?alt=media&#x26;token=6a7174e2-ffa2-47a3-9d03-310240080923" alt=""><figcaption><p>FAQ List</p></figcaption></figure>

You will also find the faq list page if you click in the `FAQs` menu from the left sidebar section.

### Create <a href="#create" id="create"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FRyfoWUMQgWqPH2OzNDoe%2F27_faqs_new_create.png?alt=media&#x26;token=39d5675c-0693-4ce2-8ad7-30c04d825b96" alt=""><figcaption><p>Create a new FAQ</p></figcaption></figure>

You would be able create a new FAQ item if you click on the `Create New` button on the FAQ list page.

Fill out the required fields and click on the `Create FAQ` button to create a new FAQ item.

### Update

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2F3CIhxOr96PMFbui3Nut9%2F26_faqs_modify_update.png?alt=media&#x26;token=2cd37520-62e2-4333-8251-f2d5059a8115" alt=""><figcaption><p>Update FAQ</p></figcaption></figure>

You can update an existing FAQ item easily with select/click on any FAQ item from the FAQ list page.

### Delete <a href="#delete" id="delete"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FOxWRc0XdckwALbhQaPsR%2F28_faqs_delete.png?alt=media&#x26;token=d83ba89d-7a62-4229-815b-ae69f0253e43" alt=""><figcaption><p>Delete a FAQ Item</p></figcaption></figure>

Go to the FAQ details page with clicking on a particular FAQ item. Then click on the "Delete" button to delete that particular FAQ.
