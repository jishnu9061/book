# Knowledge Base

Knowledge base is more than a blog post where customer would be able get information based on the particular topics. You would be able to manage knowledge base here. You will be able find the Knowledge Base manage page on the following admin link.\
`http://website.com/dashboard/knowledge_base`

### List <a href="#list" id="list"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2F0GwOip5of2K8dPGv3lDf%2F33_kb_list.png?alt=media&#x26;token=2d00898e-2656-4bb0-a27d-46d4b45165fa" alt=""><figcaption><p>Knowledge Base List</p></figcaption></figure>

You will also find the knowledge base list page if you click in the `Knowledge Base` menu from the left sidebar section. Clicking on a particular Knowledge Base item will go to the details page where you would be able to modify/delete that.

### Create <a href="#create" id="create"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2Fn29elXHlX1Pl1QT4rP4R%2F36_kb_create.png?alt=media&#x26;token=ba1037a6-59bf-428d-88c2-c51f665c7dbe" alt=""><figcaption><p>Create a new knowledge base item</p></figcaption></figure>

You would be able create a new knowledge base item if you click on the `Create New` button on the knowledge base list page.

### Update <a href="#update" id="update"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FGlqQsPaN8Y60aZ5RGlri%2F34_kb_details.png?alt=media&#x26;token=7ebeeae2-59d0-4b0c-a338-4cf785767e8e" alt=""><figcaption><p>Update a Knowledge Base Item</p></figcaption></figure>

You can update an existing knowledge base item easily with select/click on any knowledge base item from the knowledge base list page.

### Delete <a href="#delete" id="delete"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FJETI0EBj7ThJvaiwME11%2F35_kb_delete.png?alt=media&#x26;token=b4a6803c-4098-486c-88f0-0cde062b20e1" alt=""><figcaption><p>Delete a knowledge base item</p></figcaption></figure>

Go to the knowledge base details page with clicking on a particular knowledge base item. Then click on the "Delete knowledge base" button to delete that particular knowledge base.
