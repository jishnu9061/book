### Create <a href="#create" id="create"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FVNW2cUuKe28KGz9eR5jS%2F56_users_create.png?alt=media&#x26;token=55f218e2-6a89-4bd3-87b1-4dc15394a774" alt=""><figcaption><p>Create a new user</p></figcaption></figure>

You would be able create a new User item if you click on the `Create New` button on the users list page.

### Update <a href="#update" id="update"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2Fz6o45zfnkweAfDWif3fb%2F54_users_update.png?alt=media&#x26;token=03dafbc3-3b26-4bdf-96d3-de4c10829d70" alt=""><figcaption><p>Update an user</p></figcaption></figure>

You can update an existing user item easily with select/click on any user item from the user list page.

### Delete <a href="#delete" id="delete"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FKAb8K9beI5GAZNRv0WoR%2F55_users_delete.png?alt=media&#x26;token=9d6a0713-ddcf-46b8-b01b-0c562618e43f" alt=""><figcaption><p>Delete an user</p></figcaption></figure>

Go to the user details page with clicking on a particular user item. Then click on the `Delete User` button to delete that particular User.
