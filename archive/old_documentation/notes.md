# Notes

Notes is an additional modules for the HelpDesk. It works like Google Keep. Notes module will help you to store any information you want to save for the further use. You would be able to manage your notes here. You will find that notes page on the following URL. `http://website.com/dashboard/more/notes`

### List <a href="#list" id="list"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FdJpNnFoBJUz704INEjhz%2F41_notes_list.png?alt=media&#x26;token=b30bd298-d589-4858-a25f-c1a334cb1bbd" alt=""><figcaption><p>Note list</p></figcaption></figure>

You will find the note list page if you click in the `More > Notes` menu from the left sidebar section.

### Create <a href="#create" id="create"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FsD4Al06ztVmzvjBrkzQp%2F43_notes_new.png?alt=media&#x26;token=3c7628d7-944d-435a-a5c3-8e30c567d9af" alt=""><figcaption><p>Create a new note</p></figcaption></figure>

You would be able create a new note if you click on the `+` icon on the note list page.

### Update <a href="#update" id="update"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FQrSZyt0KyHOnOg8FLaS3%2F42_notes_update.png?alt=media&#x26;token=fbe5cf24-adbe-4665-888f-84cc3f812321" alt=""><figcaption><p>Update an existing note</p></figcaption></figure>

You can update an existing note item easily with select/click on that note from the list page.

### Delete <a href="#delete" id="delete"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2F257o3chwWqWYLoRrrZkk%2F44_notes_delete.png?alt=media&#x26;token=61760ff4-1707-4b11-9706-4383b6723674" alt=""><figcaption><p>Delete a note</p></figcaption></figure>

Click on a particular note, it will show with a popup dialogue. Then click on the `Delete note` button to delete that note.
