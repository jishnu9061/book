# Organizations

Organizations is an additional module where you can add your essential contacts that you need. You would be able to manage your organizations here. You will find that organizations page on the following URL. `http://website.com/dashboard/more/organizations`

### List <a href="#list" id="list"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FamLc9mn1qskoGvCCHfeX%2F49_organizations_list.png?alt=media&#x26;token=caff0a63-3b9d-402b-bfc2-fb57aeba6b2d" alt=""><figcaption><p>Organization list</p></figcaption></figure>

You will find the organization list page if you click in the `More > Organizations` menu from the left sidebar section.

### Create <a href="#create" id="create"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FsNBmDX7hPUCr0Prbj2In%2F50_organizations_create.png?alt=media&#x26;token=3d98e686-8bd1-4d90-bd08-2327524d9dbd" alt=""><figcaption><p>Create organization item</p></figcaption></figure>

You would be able create a new organization item if you click on the `Create Organization` button on the FAQ list page.

### Update <a href="#update" id="update"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2Fsei2o6jPPgetlhkzEpMs%2F52_organizations_update.png?alt=media&#x26;token=636f041a-c5f0-4124-8aeb-3afc1f6f340b" alt=""><figcaption><p>Update Organization</p></figcaption></figure>

You can update an existing Organization item easily with select/click on any Organization item from the list page.

### Delete <a href="#delete" id="delete"></a>

<figure><img src="https://3304868254-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F7fqmXAeiEs06SyjI9x4p%2Fuploads%2FgTUxZIc0Kqm9j7SPCEhv%2F51_organizations_delete.png?alt=media&#x26;token=450e1ef4-2a51-4a6d-8323-197e18715d65" alt=""><figcaption><p>Delete organization</p></figcaption></figure>

Go to the Organization details page with clicking on a particular organization item. Then click on the `Delete Organization` button to delete that Organization.
